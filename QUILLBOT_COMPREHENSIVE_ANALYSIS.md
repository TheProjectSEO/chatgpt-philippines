## 9. Common Design Patterns Across QuillBot

- Text: Dark gray, black

- Sans-serif fonts throughout

- Hierarchical sizing (48px → 16px)

### Layout Principles
1. **Container System**: Max-width ~1200-1400px
2. **Responsive Grid**: 12-column system
3. **Whitespace**: Generous padding (64-80px sections)
4. **Cards**: Rounded corners (8-12px), subtle shadows
5. **Buttons**: Large, high-contrast CTAs

### Navigation Pattern
**Desktop Navigation** (inferred):
- Logo left
- Tools menu (dropdown)
- Premium link
- Login/Sign Up right
- Sticky header

**Mobile Navigation**:
- Hamburger menu
- Drawer-style menu
- Sticky CTA button
- Bottom navigation bar

### Content Patterns
1. **Hero Formula**:
   - Attention-grabbing headline (H1)
   - Supporting subheadline
   - Primary CTA
   - Visual or demo area
   - Social proof below fold

2. **Feature Section Formula**:
   - Section headline (H2)
   - 3-4 feature cards

   - "Learn More" links

3. **Pricing Formula**:
   - 3 tiers (Free, Premium, Enterprise)

   - Annual savings highlighted
   - Feature comparison table below

4. **Social Proof Formula**:
   - Company logo strip
   - Trustpilot rating widget
   - User testimonials (quotes + photos)
   - Usage statistics (numbers)

### Interactive Elements
**Common Interactions**:
- Hover effects on buttons (darken/lighten)
- Smooth scrolling to sections
- Modal overlays for detailed info
- Toast notifications for actions
- Loading spinners for processing
- Success/error feedback

### Conversion Optimization
**CTA Hierarchy**:
1. Primary: "Try Now", "Get Started", "Upgrade"
2. Secondary: "Learn More", "See Features"
3. Tertiary: Text links in descriptions

**Placement Strategy**:
- Above the fold

### Responsive Breakpoints
- Mobile: < 640px (1 column)
- Tablet: 640-1024px (2 columns)
- Desktop: > 1024px (3-4 columns)

### Performance Optimization
- Lazy loading images
- Progressive loading

**Breakpoint Strategy**:
```css
/* Mobile First Approach */
/* Base styles for mobile (< 640px) */
.element {
  padding: 16px;
  font-size: 14px;
}

/* Tablet (≥ 640px) */
@media (min-width: 640px) {
  .element {
    padding: 24px;
    font-size: 16px;
  }
}

/* Desktop (≥ 1024px) */
@media (min-width: 1024px) {
  .element {
    padding: 32px;
    font-size: 18px;
  }
}

/* Large Desktop (≥ 1280px) */
@media (min-width: 1280px) {
  .element {
    padding: 40px;
    font-size: 20px;
  }
}
```
